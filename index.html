Mayhem

Expanding on the previous chat app Mayhem is an end to end encrypted group messaging service utilising NIO channels and selectors for high performance with many client connections.

  • End to end encryption via Sender Key system (similar to WhatsApp) to minimise bandwidth usage.
  • Public Private and Sender keys generated each session.
  • High performance with NIO Channels and concurrency.
  • Improved UI using FlatLAF.
